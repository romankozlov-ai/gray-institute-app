# Gray Institute App - Physio Testing Platform

AI-powered physiotherapy assessment platform with muscle testing, progress tracking, and PDF reporting.

## Features

- **Muscle Group Testing** - Comprehensive assessment of all major muscle groups
- **Multi-language Support** - English & Ukrainian
- **Progress Tracking** - Local storage with persistent results
- **PDF Export** - Professional assessment reports
- **Media Library** - Photo & video demonstrations for each test

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- next-intl (i18n)
- jsPDF (PDF generation)
- LocalStorage API

## Getting Started

```bash
npm install
npm run dev
```
